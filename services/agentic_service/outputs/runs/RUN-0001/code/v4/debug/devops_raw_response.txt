# AutoForge Shop

AutoForge Shop is a modular monolith e-commerce application built using the MERN stack.

## Prerequisites

- Docker and Docker Compose installed on your machine.
- Node.js and npm installed for running backend and frontend services.

## How to Run with Docker Compose

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/autoforge-shop.git
   cd autoforge-shop
   ```

2. Start the application using Docker Compose:
   ```sh
   docker-compose up --build
   ```

## How to Run Backend Manually

1. Navigate to the backend directory:
   ```sh
   cd backend
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Set environment variables (create a `.env` file):
   ```
   MONGODB_URI=mongodb://localhost:27017/autoforge_shop
   PORT=9000
   ```

4. Start the backend server:
   ```sh
   npm start
   ```

## How to Run Frontend Manually

1. Navigate to the frontend directory:
   ```sh
   cd frontend
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Set environment variables (create a `.env` file):
   ```
   VITE_API_URL=http://localhost:9000
   PORT=5173
   ```

4. Start the frontend development server:
   ```sh
   npm run dev
   ```

## Environment Variable Setup

- `MONGODB_URI`: MongoDB connection string.
- `PORT`: Port number for backend service.

## Useful URLs

- Backend: [http://localhost:9000](http://localhost:9000)
- Frontend: [http://localhost:5173](http://localhost:5173)
